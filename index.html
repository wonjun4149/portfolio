웹툰 요약 및 추천 서비스

프로젝트 개요

OCR(Optical Character Recognition) 기술과 ChatGPT API를 결합하여 웹툰의 텍스트를 추출하고 요약하며, 사용자의 선호도를 기반으로 유사한 웹툰을 추천하는 지능형 서비스입니다.

주요 기능

• 웹툰 이미지에서 텍스트 자동 추출 (OCR)
• ChatGPT API를 활용한 회차별 스토리 요약
• 웹 크롤링을 통한 웹툰 데이터 수집 및 업데이트
• 사용자 선호 장르/스타일 분석
• 머신러닝 기반 개인화된 웹툰 추천
• CUDA를 활용한 GPU 가속 처리
• 실시간 텍스트 처리 및 데이터베이스 저장

기술 스택

• Backend: Python, PHP
• Frontend: HTML, CSS, JavaScript
• Database: MySQL
• AI/ML: ChatGPT API, Tesseract OCR
• GPU Computing: CUDA
• Data Collection: Web Crawling
• OS: Ubuntu

담당 역할

• 데이터베이스 설계 및 구축
• 웹 서비스 백엔드 개발
• OCR 모듈 통합 및 최적화
• ChatGPT API 연동 및 프롬프트 엔지니어링
• 웹 크롤링 시스템 구축
• CUDA를 활용한 성능 최적화
